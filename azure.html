Let’s start some design systems

Client: Xbox (Microsoft) Agency: TEKsystems, Inc

Project Overview

Xbox Game Pass (XGP) is a SAAS game library operated by Xbox. Azure Playfab is a backend game management system for independent developers helping them manage accounts, users, in game assets and inventory, and timed events and campaigns, etc.

As a designer in the Xbox Design Studio, I worked on UI and visual explorations of various layout for the XGP PC app, before moving onto starting a library for all the UI components in production. This followed with the start of a design library of the then-recently acquired Playfab. This required a full site audit with documentation of the UI components and UX patterns. Follow-on work included recommendations for what components/patterns required consolidation or redesign for accessible/inclusive design.
